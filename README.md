# 핸드메이드스토리

수제상품 판매를 위한 핸드메이드 마켓 프로젝트입니다.

1차에서는 `Spring Boot + Thymeleaf`로 회원, 상품, 장바구니, 주문 기능을 먼저 구현하고 있습니다.  
이후 2차에서는 프론트엔드를 `React`로 옮기고 인증 방식도 다시 정리할 예정입니다.

## 프로젝트 목표

- 회원, 상품, 장바구니, 주문 기능 구현
- 세션 기반 인증과 네이버 OAuth2 로그인 적용
- 1차 MVP 완성 후 프론트엔드와 인증 구조 리팩토링

## 프로젝트 단계

### 1차 MVP

- `Spring Security + Session` 기반 인증
- `Thymeleaf` 기반 화면 구성
- 상품 조회, 상세, 등록, 관리
- 장바구니 조회, 추가, 수량 변경, 삭제
- 주문/배송 도메인 기본 구조 구현
- 서버에 배포해서 실제로 동작 확인
- `Swagger(OpenAPI)` 기반 API 문서화 추가 예정
- CI/CD 및 무중단 배포 구현

### 2차 고도화 예정

- 프론트엔드를 `React` 기반으로 리팩토링
- 인증 방식을 `JWT` 기반으로 전환
- 프론트/백 분리 구조로 정리
- 리뷰, 커뮤니티 등 확장 기능 추가

## 참여 인원 및 역할

- 참여 인원: `2명`
- 1차 MVP 담당: 백엔드와 `Thymeleaf` 기반 화면 구현
- 2차 고도화 계획: `React` 기반 프론트엔드 리팩토링과 함께 백엔드 정리, API 연동, 일부 화면 전환 작업 진행 예정

## 현재 구현 범위

### 회원/인증

- 로컬 회원가입/로그인
- 세션 기반 인증 처리
- 네이버 OAuth2 로그인
- 로그아웃 및 사용자 정보 조회

### 상품

- 상품 목록 조회
- 상품 상세 조회
- 카테고리별 조회
- 상품 검색
- 상품 등록
- 판매자 본인 상품 목록 조회
- 상품 삭제 및 재고/판매 수량 반영

### 장바구니

- 내 장바구니 조회
- 상품 담기
- 수량 변경
- 단일 상품 삭제
- 장바구니 전체 비우기

### 주문/배송

- 주문 생성, 조회, 수정, 상태 변경, 삭제
- 배송 도메인 기본 CRUD

## 기술 스택

### Backend

- Java 17
- Spring Boot 3.4.3
- Spring Security
- Spring OAuth2 Client
- Spring Data JPA
- Thymeleaf

### Database

- PostgreSQL

### Build / Test

- Gradle
- JUnit 5
